INTRODUCCIÓN

Este documento contiene información necesaria para el correcto uso del Sistema de Interoperabilidad de los Operadores de Justica (SIOJ) - Modulo de Comisaria, por parte del usuario, siendo este documento un manual de usuario final.

OBJETIVOS

El objetivo del presente documento es de ser una guía al usuario para mostrar la forma correcta del uso del aplicativo.

Registro de Denuncia

En este apartado el INSTRUCTOR, podrá registrar, buscar y generar la denuncia verbal y todos los documentos pertenecientes a la misma.

Podrá escoger entre, registrar una NUEVA denuncia. Y BUSCAR una denuncia, donde podrá buscar por PERIODO, ESTADOS, NUC y/o NOMBRES de las partes.

Adicionalmente podrá visualizar el documento de la DENUNCIA y la TRAZABILIDAD, es decir el donde se encuentra actualmente la DENUNCIA.

Para visualizar la denuncia debe generar la CARPETA POLICIAL, al terminar de registrar la DENUNCIA. Para visualizar la TRAZABILIDAD haga clic en el icono en la columna TRAZABILIDAD. La TRAZABILIDAD se visualizara.

Adicionalmente podrá visualizar el documento de la DENUNCIA y la TRAZABILIDAD, es decir el donde se encuentra actualmente la DENUNCIA.

Documentos

Seleccione DOCUMENTOS Se visualizará la pantalla de DOCUMENTOS, donde podrá registrar los documentos faltantes, y firmar electrónicamente. Para acceder a esta sección debe generar NUC, es decir guardar los cambios.

4.2.1. Informe Policial En esta sección el COMISARIO, podrá redactar el informe policial, donde registra: - Antecedentes que motivan la intervención policial. - Investigaciones realizadas. - Análisis de los hechos. - Situación de las partes implicadas. - Anexos. Seleccione el ICONO en la columna LLENAR FORMULARIO.

El sistema muestra el formulario de INFORME POLICIAL.

Para visualizar el documento haga clic en el icono de PDF en la columna.

El documento se visualizara, proceda a descárgalo y fírmalo con el software REFIRMA.

Seguridad

Cambio de Clave

Esta opción permite cambiar la clave de cualquier usuario del sistema. Para acceder seleccione el icono , CAMBIO DE PASSWORD, se visualizara la ventana de CAMBIO DE CLAVE.

Una vez ingresado los datos y aceptar los cambios el sistema mostrara un mensaje de confirmación.

PREGUNTAS

¿En qué formatos puede exportar el reporte?

El formato de visualización de constancias es PDF.

¿Puede utilizar cualquier navegador web para acceder al aplicativo?

Si puede utilizar cualquier navegador, sin embargo puede darse el caso en que la presentación de la página no se vea correctamente, es por ello que se recomienda el uso de Internet Explorer 8 o superior y/o Google Chrome y/o Mozilla Firefox.
